This course is really great in a lot of ways, but in its current form it is not a very good class (though it is easy, if that's what you're looking for). The subject material is fairly interesting and it covers a ton of topics. However, its current format (80 minute lecture x 80 minute precept (that can even be back to back) does it a disservice. Additionally, it tries to cover too much material. What's more, we spent much of the class time discussing gender and race, and while these are crucial topics, they took up too much class time for what the class was billed. Also: having 3 professors rotate lectures disrupted any momentum the class got going, and this was especially tough because it only met once a week. If you are taking for the certificate or for an easy class: definitely take it. If you just want an elective: there are better options unless this course gets a makeover.
