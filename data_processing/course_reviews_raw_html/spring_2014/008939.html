If you are not a CBE major, please don't take this class--it is a huge time commitment and you won't get much out of it. CBE majors--START EARLY! As soon as you get some data, analyze it to the best of your ability. Start error analysis early as well. Ask a professor if you are confused. Don't forget error bars, and don't forget uncertainty intervals. Although this already seems like a lot to do, make sure to analyze your data that you collect in each lab before the next one. This will help you make more informed measurements as labs go on. If you do core lab right, it can be one of the most worthwhile experiences of Princeton. If you don't, you'll end up hating every minute of your life.
