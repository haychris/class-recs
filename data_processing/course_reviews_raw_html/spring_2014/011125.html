I am honestly very conflicted about how to review this class. On the one hand, the workload is very low, the material is interesting, and Nelson provides pertinent reading material for those who are actually interested in delving deeper into a certain topic. As a second semester senior who took this class out of interest, I could choose how much or how little effort to put in, depending on what was being covered in class. On the other hand, our class had ~80 students and was all lecture based. It would have been much better had there been only ~30 students and if it was discussion-based. Moreover, it is very difficult to get a good grade not because the class is hard, but because the grading for problem sets is rather arbitrary. It doesn't require you to use any knowledge from class, but rather requires you to be good at wikipedia searches and if you don't mention certain details, you will get points off. Now, this wouldn't be a problem if the averages on problem sets weren't ~90% and if problem sets didn't make up almost all your grade. The only other grade is a final project. Overall, I know Nelson has worked hard to plan this course and she has done a great job of choosing which topics to include, but the class size needs to be smaller, grades need to be based more on class discussion, and grading of problem sets needs to be less nitpicky.

If you want to know about the most important diseases in America (and, to a certain extent, the world) today, you should definitely take this class. We covered heart disease, cancer, diabetes, and tuberculosis, which each affects thousands to millions of individuals (or, you could argue, billions in the case of tuberculosis). This class is good if you want to come to the end of a class and feel like you've learned something worthwhile. Note: The problem sets are hard, so go to office hours to figure out the types of things they're looking for. Note II: You get to play with three different organs over the course of the semester (all from pigs): a pancreas, a pair of lungs, and a heart. It is really cool. Note III: The course project at the end of the semester is great because you can do it on whatever disease you want, so it really encourages you to get an in depth understanding of a disease that interests you. So if you want to learn some very useful knowledge about diseases---but also about how the healthy body works---take this course! It is a great primer on the mechanisms of the body.
